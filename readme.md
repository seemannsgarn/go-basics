## Verbs

%x,%o,%b    // integer 16,8,2

%f,%g,%e    // float: 3.141593, 3.141592653589793, 3.141593e+00

%t          // bool

%c          // rune(Unicode)

%s          // string

%q          // output like "abc" or 'c'
